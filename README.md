mruby-env
=========

### To build:

Prerequisites:

    * mruby
    * libc

    activate GEMs in *build_config.rb*
    * conf.gem :git => 'https://github.com/iij/mruby-env.git', :branch => 'master'
    ruby ./minirake

### To run the tests:

    ruby ./minirake test
